What is Parcelfile?

Parcelfile is a python module that does file processess in python programs. It can help with complex python programs that involve multiple functions that can be done with files.

Why was Parcelfile made?

This project was mainly built to be helpful in python projects, but it's creation was not just for it's helpful features. It can be an enjoyment to accomplishing the creation of of a python module.

How does Parcelfile work?

Parcelfile uses a combination of python commands, file options, and other modules to easily perform quick file funtions. The commands are made to be simple, so they won't need to be practiced.



How do I install Parcelfile?


Windows installation

Note: before installation on any system, make sure you have python preinstalled


To install Parcelfile onto a windows computer, use the command line pip install parcelfile

To import Parcelfile, use from parcelfile import * to import all the syntax.

MacOs/Unix installation

To install Parcelfile on a MacOs or Unix computer, use the command line python pip3 install parcelfile

Importing is the same as Windows
